# indexs-bsports-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面是静态资源，不依赖于特定的后端服务或域名。

## 项目简介

`indexs-bsports-resource` 是一个集中存放 HTML 页面的仓库。每个页面都是独立、自包含的，可以用于展示、测试或作为某些资源的离线版本。仓库本身不针对任何具体网站或业务，仅作为静态文件的存储和分发点。

## 目录结构

```
├── index.html          # 主入口页面（如有）
├── pages/              # 存放各个独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：包含所有独立的 HTML 文件，每个文件均可单独使用。
- **assets/**：存放页面共用的静态资源，如图片、样式表和 JavaScript 文件。

## 页面归档说明

- 每个 HTML 页面均为独立归档，可直接在浏览器中打开运行。
- 页面之间不存在依赖关系，除非特别说明。
- 资源文件路径使用相对路径，确保在本地或静态服务器上均可正常工作。

## 维护说明

- 本仓库内容会不定期更新，以添加新的页面或修正已有页面。
- 提交新页面时，请确保所有资源引用正确，且页面可独立运行。
- 不建议在页面中嵌入外部不可控的链接或资源，以保证归档的稳定性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 目录下的任意 `.html` 文件。
3. 或使用任意静态服务器（如 Python `http.server`）托管整个目录。
